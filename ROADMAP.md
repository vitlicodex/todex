# Roadmap

This roadmap is intentionally conservative. The app should remain native, local-first, and lightweight.

## Near Term

- Developer ID signing and notarized release artifacts.
- More automated tests for API response parsing and security edge cases.
- Better export formats for numeric reports.
- Optional menu customization for exact vs compact values.

## Later

- Native SwiftUI settings window.
- More granular privacy controls for source paths in reports.
- Optional notification thresholds.
- Signed release update feed.

## Non-Goals

- Storing prompt contents.
- Sending local usage telemetry to a third-party service.
- Replacing Codex's own permission system.
- Heavy Electron-style desktop shell.
